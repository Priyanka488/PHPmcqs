Forms

Forms are used to get input from the user and submit it to the web server for processing. A form is used to get input from the user and to submit it to the web server for further processing. In HTML , we have form tag that contains various graphical user interface items such as input box, check boxes radio buttons etc.


The form is defined using
< form > (Starting tag)
< /form >(Ending tag)

Creating a Form

We will be using HTML to create our form. The minimum info that should be present in the code is:

  • Opening and closing form tags.
  • Form submission type POST or GET.
  • Name of the PHP file where the data is sent for processing.
  • Input fields such as input boxes, text areas, buttons,checkboxes etc.
  • The PHP superglobals $_GET and $_POST are used to collect form-data.


Let’s walk down line by line and understand the code :

< form action="welcome.php" method="post">

  • < form > is the html tag to that contains various input GUI
  • action=”welcome.php” specifies that the input entered by the user will be sent to the file named as welcome.php
  • method=”post” specifies that submission is of type “post”.The form data is sent through HTTP POST method.

Name: < input type="text" name="fname">

  • “Name:” is the text displayed on the screen on the input box, generally acting like a placeholder or a hint.
  • type=”text” , specifies that the input is of type text.
  • name=”fname” acts like a variable associated with the input,this variable (fname) will be used in the php script to retrieve the input name.

< input type="submit">
< /form >

  • type=”submit” indicates of a button, used to submit the input values by the user.
  • < /form > is the closing tag of the HTML form.

PHP Script

Welcome < ?php echo $_POST["name"]; ?>
Your email address is: <?php echo $_POST["email"]; ?>


  • PHP tags open and close as follows :

    • < ? php (Starting tag)
    ? >(Ending tag)

  • "echo" is a php command used to display content on screen.
  • $_POST["fname"] - Here , $_POST is a superglobal array consiting of all the values sent through HTTP POST method.
  • Note we use the variable associated with the attribute name in HTML as the key to retreive the variable.
